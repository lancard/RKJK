------
설치법
------
본 시너리는 MS FS2004 전용으로 제작되었으며 타 버전과의 호환성 여부는
장담할 수 없습니다. 가급적 MS FS2004에서 시너리를 설치 및 실행해주시기 바랍니다.

파일의 압축은 MS FS2004의 Addon scenery폴더에 임의의 폴더를 생성하신 후
압축을 풀어주시고

시너리 등록은 수동등록인 관계로, 본 시너리를 설치한 후, MS FS2004를 실행하고
시너리 라이브러리에서 군산공항 시너리를 등록한 후, 사용해주시기 바랍니다.
